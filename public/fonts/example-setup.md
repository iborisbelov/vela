# Как добавить локальные шрифты

1. Создайте папку: `public/fonts/`
2. Поместите файлы шрифтов (.woff2, .woff, .ttf)
3. Добавьте @font-face в src/index.css
4. Обновите tailwind.config.ts

Пример структуры:

```
public/
  fonts/
    MyFont-Regular.woff2
    MyFont-Bold.woff2
    MyFont-Light.woff2
```
